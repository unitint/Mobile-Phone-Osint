# 📱 Mobile Phone OSINT Tool

## 🕵️ What is this?
A powerful OSINT (Open Source Intelligence) tool to investigate phone numbers across multiple platforms including social media, search engines, and public records.

## 🚀 Features
- 🔍 Search Google for ALL variations of a phone number
- 📘 Check Facebook presence
- 📸 Check Instagram presence
- 💬 Check WhatsApp presence
- ✈️ Check Telegram presence
- 🐦 Check Twitter/X presence
- 📄 Search Google Docs and PDF files
- 📋 Generate comprehensive reports
